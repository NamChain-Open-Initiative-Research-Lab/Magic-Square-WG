# Magic Square Working Group (MSWG)
![](https://img.shields.io/badge/WG-Magic_Square-gold)

This Working Group Focuses on understanding the Magic Squares and find suitable applications of them in Modern Cyber Security landscape. In addition, the Working Group also shall create Recreational Magic Squares.

**Project Start Date:** Feb 21, 2021

#### Religious Context
